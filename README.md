# Self-Inquiry Tool (The Work)

A digital reflection worksheet based on the "The Work" by Byron Katie. This tool provides a structured interface for the process of self-inquiry, helping users investigate stressful thoughts through four specific questions and a series of "turnarounds."

* **Privacy First:** As a static HTML file, all reflections stay local to the user's browser; no data is stored or transmitted.
